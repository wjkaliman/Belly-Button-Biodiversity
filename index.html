Belly Button Biodiversity Dashboard

Welcome to the Belly Button Project. This project used JavaScript, Ploty, Python, D3, JSON, and HTML to explore the Belly Button Bacteria Biodiversity DataSet. The data set was read in JSON format and brought in by D3. I built 2 graphs using JavaScript and Plotly

When you select a Test Subject ID No., the Metadata is displayed in the 2 graphs and the Demographic info . Click a few ID's to see the changes.The graphs will also resize when the page is resized.
